# JJBIC
Mobile inventory management software for everyday use in the average household. 
This repository holds the code of a group project in my Software Engineering course at Saint Vincent College.
This app is being devloped in the Flutter IDE, and built using the programming language Dart. 
We are developing an Android mobile app that runs as an Inventory Management Software (IMS). This allows the everyday person to manage their house's inventory just by utilizing our app. Anyone can add items, quantities, lists, expiration dates, etc. 
The app is at the brink of creation and we hope to do more with it as the semester rolls out and perhaps beyond.
